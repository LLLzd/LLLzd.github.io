- work1.jpg - 行书《兰亭序》
- work2.jpg - 楷书《心经》
- work3.jpg - 草书《千字文》
- work4.jpg - 隶书《岳阳楼记》
- work5.jpg - 篆书《道德经》
- work6.jpg - 行书《赤壁赋》

建议图片尺寸：800x600 像素或更高
建议图片格式：JPG 或 PNG

## 展览现场 (images/exhibitions/)
请在此目录下放置展览现场照片，文件名如下：
- exhibition1.jpg - 2024年北京书法展
- exhibition2.jpg - 2023年上海书法艺术展
- exhibition3.jpg - 2022年广州书法精品展

建议图片尺寸：800x600 像素或更高
建议图片格式：JPG 或 PNG

## 荣誉奖项 (images/honors/)
此目录用于存放奖杯、证书等照片
目前使用emoji图标展示，如需替换为真实照片，请在此目录下添加图片文件

## 注意事项
1. 请确保图片文件名与HTML中引用的文件名一致
2. 建议使用高质量的图片以获得更好的展示效果
3. 图片文件大小建议控制在2MB以内，以保证网站加载速度
4. 如需添加更多作品，请在HTML中复制相应的代码块并修改图片路径
