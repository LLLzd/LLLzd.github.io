# 刘从东个人书法作品展示网站

## 项目简介

这是一个高端、大气、中式美学风格的个人书法艺术官方网站，用于展示书法家刘从东的书法作品、展览现场、荣誉奖项等信息。

## 网站特色

- **中式美学设计**：采用传统中式色彩（石色、琥珀金），营造典雅大气的视觉效果
- **响应式布局**：完美适配手机、平板、电脑等各种设备
- **流畅动画**：优雅的滚动动画和过渡效果
- **图片放大预览**：点击作品可放大查看，支持键盘导航（左右箭头、ESC）
- **SEO优化**：完整的meta标签，有利于搜索引擎收录
- **动态加载**：图片通过JavaScript动态加载，页面加载速度快

## 当前图片统计

- **书法作品**：22张
- **展览照片**：19张
- **证书照片**：14张

## 技术栈

- **HTML5**：语义化标签，结构清晰
- **Tailwind CSS**：现代CSS框架，快速构建响应式设计
- **JavaScript**：原生JavaScript，无依赖，性能优异
- **Google Fonts**：使用思源宋体和马善政书法字体，体现中式书法美感

## 快速开始

### 1. 本地运行

直接在浏览器中打开 `index.html` 文件即可查看网站。

或者使用本地服务器（推荐）：

```bash
# 使用 Python 3
python3 -m http.server 8080

# 然后在浏览器访问
http://localhost:8080
```

### 2. 更新图片

直接将图片复制到 `images/` 目录下的对应文件夹：
- `images/works/` - 书法作品
- `images/exhibitions/` - 展览现场
- `images/honors/` - 荣誉奖项

图片文件名会自动作为图片描述显示在网站上。

### 3. 自定义内容

#### 修改个人信息

在 `index.html` 中搜索以下内容并替换：

- 书法家名字：搜索"刘从东"
- 联系方式：搜索"电话"、"微信"、"地址"
- 艺术简介：搜索"个人经历"、"艺术风格"、"主要成就"

#### 修改艺术格言

在 `index.html` 第75行，修改格言内容：

```html

    笔耕不辍，墨润初心
```

## 常见问题

### Q: 如何添加新的书法作品？

A: 将新图片放到 `images/works/` 目录，图片文件名会自动作为图片描述显示。

### Q: 如何修改联系方式？

A: 在 `index.html` 中搜索"电话"、"微信"、"地址"，修改对应的文字内容。

### Q: 如何添加微信二维码？

A: 将微信二维码图片放到 `images/` 目录，然后在 `index.html` 的联系方式部分添加图片标签。

### Q: 网站加载速度慢怎么办？

A: 
1. 压缩图片文件大小
2. 使用图片CDN加速
3. 启用浏览器缓存
4. 使用Gzip压缩

### Q: 如何让百度更快收录？

A: 
1. 提交网站到百度站长平台
2. 创建sitemap.xml文件
3. 增加外部链接
4. 定期更新内容

### Q: 如何部署到 GitHub Pages？

A: 
1. 将代码推送到 GitHub 仓库
2. 在仓库设置中找到 "Pages" 选项
3. 在 "Source" 下选择 "Deploy from a branch"
4. 选择 "main" 分支和 "/ (root)" 文件夹
5. 点击 "Save"，等待部署完成
6. 访问 `https://你的用户名.github.io/` 查看网站
