# 刘从东书法艺术网站

## 项目介绍

刘从东书法艺术官方网站，展示书法家刘从东的书法作品、展览活动和荣誉奖项。

## 网站结构

- **首页**：个人简介、推荐作品、最新活动
- **经典作品**：书法作品展示，包括图片和视频
- **活动资讯**：展览现场照片和视频
- **艺术简介**：个人经历、艺术风格、主要成就
- **资质证书**：获得的荣誉与证书
- **联系方式**：电话、微信、地址

## 技术特点

- **响应式设计**：适配手机、平板、电脑
- **中式美学风格**：米黄色背景，琥珀色强调色，中国风纹理
- **图片和视频支持**：支持MP4视频文件播放
- **大图预览**：点击图片可放大查看
- **SEO优化**：设置了网站标题、关键词和描述

## 图片和视频

### 作品展示 (images/works/)
- 包含书法作品图片和视频
- 支持MP4格式视频

### 活动资讯 (images/exhibitions/)
- 包含展览现场照片和视频
- 支持MP4格式视频

### 资质证书 (images/honors/)
- 包含各类证书和荣誉照片

## 如何更新内容

### 更新图片和视频
1. 将新图片/视频放入对应目录
2. 更新 `images_data.json` 文件，添加新文件信息
3. 确保文件名不包含数字后缀（如"出版物2"）

### 修改文字内容
- 编辑 `index.html` 文件，修改相应部分的文字

## 本地测试

```bash
# 启动本地服务器
python3 -m http.server 8080

# 访问网站
http://localhost:8080
```
